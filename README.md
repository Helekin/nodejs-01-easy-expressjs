## Getting Started

### Prerequisites

- Use any code editor

### Installation

1. Clone the repository to your local machine

```sh
git clone https://github.com/Helekin/nodejs-01-easy-expressjs.git
```

2. Run the application in development mode

```sh
npm run dev
```
